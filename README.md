# Teslo API

1. Clonar proyecto
```
npm i
```
2. Clonar el archivo ```.env.template``` y renombralo a ```.env```
3. Cambiar las variables de entorno
4. Levantar la base de datos
```
docker-compose up -d
```
5. Levantar
```
npm run start:dev
```
